Course Tools

Browser-based tools for previewing Canvas course content.
Files are processed locally — nothing is uploaded anywhere.

📦
IMSCC Course Viewer
Preview a full Canvas course export (.imscc / .zip) — browse modules, pages, quizzes, assignments, and discussions.
→ 📋
QTI Quiz Viewer
Preview a Canvas QTI 1.2 quiz export (.xml) — renders all question types with interactive check-answer functionality.
→

All processing happens in your browser. No data leaves your computer.
